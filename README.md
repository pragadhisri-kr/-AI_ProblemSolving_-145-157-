##  Problem 1: Warehouse Robot Planner

### Description

A robot rearranges blocks from an initial configuration to a goal configuration using valid moves.

### Algorithm Used

* State Space Search
* Breadth First Search (BFS)

### Features

* Stack-based block representation
* Valid moves (only top block)
* Step-by-step solution
* GUI using Tkinter

### Sample Output

* Move C from Stack 1 to Stack 2
* Move B from Stack 1 to Stack 2

##  Problem 2: Smart Emergency Response System

### Description

Finds the best route for emergency services using AI search.

### Algorithm Used

* A* Search Algorithm

### Features

* Select emergency type
* Finds nearest service
* Displays optimal path and cost
* GUI using Streamlit

### Sample Output

* Path: A → B → E → F
* Cost: 6
